# Selected Wear Store

Fertiges Next.js-Projekt für einen schlichten grau-weißen Fashion-Store.

## Lokal starten

```bash
npm install
npm run dev
```

Dann im Browser öffnen:

```bash
http://localhost:3000
```

## Deployment mit Vercel

1. Dieses Projekt als ZIP entpacken.
2. Auf GitHub ein neues Repository erstellen.
3. Alle Dateien hochladen.
4. In Vercel das Repository importieren.
5. Deploy klicken.
